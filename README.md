# Fokontaniko

Fokontany management

## Requirements:

```
- php v^7.4
- nodejs && npm || yarn
```

## Installation :

```
- git clone your_fork.git
- cd Fokontaniko 
- composer install
- yarn install
```

- Update the database configuration in `.env`

## Database:

```
- bin/console d:d:c
- bin/console d:s:u -f
```

## Command to create super admin user :

`- bin/console app:create:user`
